SaaS

The software as a service (SaaS) comes with the following tiers, with one yearly price:

Standard

£949/yr

Includes:

  • Up to 10,000 user entries
  • Unlimited API calls
  • 10 consent templates & 10 consent services
  • 3 client consent service administrators
  • Online support.
  • One days worth of consulting to help with deployment
Contact sales

Pro

£8,999/yr

Includes:

  • Up to 100,000 user entries
  • Unlimited API calls
  • Unlimited templates & Unlimited services
  • 10 client consent service administrators
  • Standard support Monday to Friday
  • Two days of consulting to establish your client consent service
Contact sales

Enterprise

(On-Premise / Own Cloud)

Licence to deploy and host Consentua on your own premise servers or cloud service

  • This provides an unlimited number of hosts in the repository
  • Unlimited API Calls
  • Unlimited number of client system administrators
  • Five days free consulting to establish and set up and support your new consent services
  • Standard support Monday to Friday - 0800 to 1800 UTC
